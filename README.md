# 알고리즘 문제풀이

## 파일명 규칙
- P + 문제번호

## 태그
> main 첫번째 줄 주석에 문제와 관련된 태그를 명시한다. (향후 검색 목적)
> ex1) // #BFS #소수 
> ex2) // #미정 - 범주를 잘 모를때.
> 아래에는 1개 이상의 문제에 달린 태그를 모아 놓고자 한다.
> 다시 볼만한 문제는 `리뷰` 태그를 단다. 
> 시간이 너무 오래걸려서 못풀고 강의에서 제공한 답을 본 문제의 경우, `답지` 태그를 단다.
```
#미정, #리뷰, #답지
#구현
#형변환
#브루트 #완탐 #백트래킹 #NM
#분할정복 #그리디 #우선순위큐 #DP #이분탐색  
#BFS #BFS(최단거리) 
#DFS #플러드필 
#다익스트라 #플로이드 #경로복원
#입출력효율 #메모이제이션 
#정규표현식
#소수 #누적합 #나머지연산
#TSP 
#트리 #트리순회 #트리DP(아직 없음)
#스택 #연결리스트

```
